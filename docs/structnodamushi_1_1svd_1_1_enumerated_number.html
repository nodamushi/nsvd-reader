Detailed Description

Decode and represent enemeratedValue.value.

format:
  • [0-9]+ :decimal
  • 0x[0-9A-Fa-f]+ :hexadecimal
  • 0b[01x]+ :binary
  • #[01x]+ :binary
    binary format is able to contains `‘don't care’' bit information.
    See also
    http://www.keil.com/pack/doc/CMSIS/SVD/html/elem_registers.html#elem_enumeratedValue
